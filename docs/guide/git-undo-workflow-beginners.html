초보자를 위한 GitHub 되돌리기 흐름

버튼으로 한 단계씩 진행하며 커밋·푸시·머지·소스를 어떻게 안전하게 되돌리는지 눈으로 확인하세요. 명령은 모두 실제 실행으로 검증했습니다(git 2.43 기준).

로컬 (내 컴퓨터)
·
원격 (GitHub)

읽는 법 — 회색 사각형은 커밋 1개, 파란 사각형은 “이번 단계에서 생긴/관련된 커밋”, 점선 사각형은 “되돌려져 사라진 커밋”입니다. 파란 글씨는 “지금 일어나는 동작/현재 위치(HEAD)”만 표시합니다.
안전 원칙 — reset --hard·--force는 아직 push 안 한 로컬에서만. 이미 원격에 올라갔고 협업 중이라면 이력을 보존하는 git revert를 쓰세요.
